appenginepythondemo
===================

=Google App Engine Python Demo=
I build this simple application to test how app engine works. App uses Google User API for login. Logged in users can upload Images, Audios, Videos etc.
All the files are stored in Google BlobStore and the records are kept in Cloud SQL database. I am storing the Blob Key values in the DB to link the blobfiles.
